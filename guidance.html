Guidance Page

To test the prototype please pay attention to the following:

  • Please test as usual. Search how you would search on PubMed.
  • You can use the search in either Enligh or Japanese.
  • To validate the prototype, it is necessary to pat attention to the generated summaries.
  • This webapp can be tested on mobile as well as on desktop.

Limitations:

  • The code of this webapp is not yet optimised. Therefore, requests can take 20 - 40 seconds.
  • The webapp only searches freely available articles, which are listed on PubMed Central.
  • Japanese search is translated into English, should not result be available, please try to search in English.
